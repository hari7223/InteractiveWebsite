This is a website designed for a Cloud Computing course assignment.

The front-end was developed using HTML+CSS & JS.
The back-end is developed using Flask(Python).

This is a full-stack Flask web application that allows users to:
	•	Register and log in
	•	Upload text files and automatically compute word count
	•	Store files securely in AWS S3
	•	Generate secure presigned URLs for downloading the file which was uploaded
	•	View user profile details including uploaded file and word count

The website is hosted on an AWS EC2 server. Please message if you want to access the website.
